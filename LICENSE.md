**❌ GUI doesn't appear after execution**
→ Your executor may not have sufficient UNC support. Switch to a Level 7
  executor such as Synapse Z, Delta, or Xeno and re-execute the script.

**❌ Orbit Player stops working mid-session**
→ The target player may have left the game or respawned with a new
  character instance. Open the GUI, deselect the target, and reselect
  them from the updated player list.

**❌ Auto Farm Players earns no resources**
→ The game may have patched the resource endpoint this script targets.
  Check this repository for an updated version and re-execute after
  updating.

**❌ Teleport puts character in the void or wrong location**
→ Use the Teleport bookmarks feature to save verified safe coordinates.
  Avoid teleporting during active combat animations, as character state
  can interfere with position updates.

**❌ ESP names and health bars flicker**
→ Toggle ESP off and back on in the GUI. If flickering persists, reduce
  your Roblox graphics quality to Level 5 or below to reduce rendering
  conflicts.

**❌ Script works on PC but crashes on mobile**
→ Download the dedicated mobile version of the script from the link
  above. The PC version uses some API calls that are not supported in
  all mobile executor environments.

**❌ "attempt to index nil" error on execution**
→ Make sure you are fully loaded into a live Medieval RTS game session
  before executing — not in the lobby or loading screen.
